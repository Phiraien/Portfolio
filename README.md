# Portfolio

> Personal portfolio website for **Phi Raien** — engineering student, AI explorer, and builder.

**Live:** [phiraien.github.io/Portfolio](https://phiraien.github.io/Portfolio/)

A clean, responsive single-page portfolio built with vanilla HTML, CSS, and JavaScript. No frameworks, no build tools, no bloat.

---

## Sections

| Section | What it covers |
|---------|---------------|
| **Hero** | Name, tagline, quick links to GitHub and contact |
| **About** | Background, current focus (AI agents, local LLMs, systems), stats counter |
| **Skills** | Languages, frontend, backend/DB, tools & AI — 4-column grid |
| **Projects** | Featured repos with tags and direct GitHub links |
| **Contact** | Email and GitHub link |

## Tech Stack

- **HTML5** — semantic structure
- **CSS3** — Flexbox/Grid, responsive breakpoints, glassmorphism nav, smooth transitions
- **Vanilla JS** — IntersectionObserver for scroll-triggered fade-in, smooth anchor scrolling

## Local Development

```bash
# Clone
git clone https://github.com/Phiraien/Portfolio.git

# Open in browser (no build step)
open index.html
```

Just open `index.html` in any browser — no server, bundler, or dependencies needed.

## Deployment

Hosted on **GitHub Pages** — auto-deploys from the `master` branch. Any push to `master` triggers a rebuild.
