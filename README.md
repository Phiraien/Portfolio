# Portfolio

Personal portfolio website for **Phi Raien** — engineering student, developer, and builder.

**Live:** [phiraien.github.io/Portfolio](https://phiraien.github.io/Portfolio/)

---

## Sections

- **Hero** — name, tagline, GitHub + contact links
- **About** — background, current focus, stats
- **Skills** — languages, frontend, backend & DB, tools
- **Projects** — featured repos with descriptions and links
- **Contact** — email and GitHub

## Tech Stack

- **React 19** + **Vite 6**
- CSS3 (Flexbox, Grid, CSS variables, dark mode via `prefers-color-scheme`)
- Vanilla JS (IntersectionObserver)

## Development

```bash
npm install
npm run dev     # dev server
npm run build   # production build
npm run deploy  # build + push to gh-pages
```
